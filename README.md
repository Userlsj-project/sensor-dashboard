# Smart Factory Virtual Sensor Monitoring System

A simulated IoT sensor monitoring dashboard built with the LAMP stack and Python Flask on Ubuntu 24.04. No physical hardware required — all sensor data is generated by a Python simulator.

---

## Features

- Real-time sensor data simulation (temperature, humidity, pressure)
- PHP dashboard with Chart.js visualization (auto-refreshes every 5s)
- Flask REST API for JSON data access
- MySQL 8.0 backend
- 3 virtual sensors: SENSOR-01, SENSOR-02, SENSOR-03

---

## Prerequisites

- Ubuntu 24.04 (or Zorin OS / VMware environment)
- LAMP stack: Apache2, MySQL 8.0, PHP 8.3
- Python 3.x

---

## Installation

### 1. Install system packages
```bash
sudo apt update
sudo apt install -y apache2 mysql-server php php-mysqli libapache2-mod-php python3 python3-pip
```

### 2. Install Python dependencies
```bash
pip3 install flask flask-cors mysql-connector-python
```

### 3. Set up the database
```bash
chmod +x db_setup.sh
./db_setup.sh
mysql -u sensor_user -psensor_pass sensor_db < setup.sql
```

### 4. Deploy the dashboard
```bash
sudo cp index.php /var/www/html/index.php
sudo systemctl restart apache2
```

---

## How to Run

**Terminal 1 — Start the sensor injector:**
```bash
python3 injector.py
```

**Terminal 2 — Start the Flask API:**
```bash
python3 api.py
```

**Open in browser:**
- Dashboard: http://localhost/
- API (last 20 rows): http://localhost:5000/api/data
- API (latest per sensor): http://localhost:5000/api/latest
- Health check: http://localhost:5000/api/health
